# 2D-Cat-And-Mouse-Game
A text-based chaser game written in C++. The player must collect all of the food in the maze while a cat roams the walls as well.
Developed in the Visual Studio 2022 IDE.

 ## Features
 - Procedurally generated maze of customizable size.
 - Enemy and collectable food spawned within a minimum distance of the player.
 - Player movement with WASD and automatic enemy movement, each with customizable speeds.
 - Game loop with win / loss states.
 - A scent trail left by the player that dissipates over time and leads the enemy to chase the player.
 - Colored console output to increase clarity over the game board.
 - FMOD Libraries added to add sound output to the application.
